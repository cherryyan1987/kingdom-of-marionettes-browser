# Kingdom of Marionettes (Browser Version)

Kingdom of Marionettes is a story-driven indie horror visual novel set in an abandoned puppet theater.

The game features a unique atmosphere, expressive characters, and multiple endings based on player choices. One of the highlights is a mysterious clown character that feels both playful and unsettling.

## Why this version?

The original game is only available on PC, which limits accessibility for many players.

This browser version allows you to experience the game instantly without installation.
